# ❤️ HeartGuard
### Preventive Insights for Heart Failure Readmission using Machine Learning

## 📌 Overview
HeartGuard is a smart healthcare web application that predicts the risk of heart failure readmission using machine learning. It helps users understand their health condition, identify risk factors, and receive personalized lifestyle recommendations.

The system also includes an AI chatbot for heart health awareness and guidance.

---

## 🚀 Features

- 🔍 **Heart Risk Prediction**
  - Predicts the probability of heart failure readmission
  - Uses trained ML model (Logistic Regression / Random Forest)

- 📊 **Interactive Dashboard**
  - Displays user health metrics
  - Visualizes predictions using charts (Chart.js)

- 🤖 **AI Chatbot**
  - Answers heart-related queries
  - Explains prediction results
  - Provides diet, exercise, and lifestyle tips

- 🎯 **Personalized Suggestions**
  - Health improvement recommendations
  - Risk reduction strategies
